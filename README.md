# 个人作品集网站（Personal Portfolio）

一个基于 Next.js + React + TypeScript 开发的个人作品集网站，用于展示个人项目、研究经历以及 AI 相关实践。

## 项目简介

本项目是我的个人作品集网站，主要用于展示个人技术能力、项目经历以及后续的 AI 相关作品。

网站采用现代前端技术栈开发，并部署在 Netlify，可进行在线访问。

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS 4
- Netlify

## 已实现功能

- 响应式页面布局
- 个人信息展示
- 项目展示
- 联系方式
- 网站部署上线

## 开发说明

本项目采用 **AI 协同开发** 的方式完成。

在开发过程中，我主要负责：

- 项目需求规划
- 页面设计与功能规划
- 技术方案选择
- 功能测试与 Bug 调试
- 项目部署与维护

Codex 作为 AI 编程助手参与了代码生成工作，我负责整体开发方向把控、需求拆解、代码调试、功能验证以及最终部署。
